# FRC-Team-Lookup 

## How to Install
- Software Needed
    - pip
    - python3
    - firefox
    
- Steps
    - 1 make sure pip is in your path
    - 2 make sure Firefox is installed and in your path
    - 3 install selenium using pip `pip install selenium`
    - 4 Type in cmd `python3 Main.py`
